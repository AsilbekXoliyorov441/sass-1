SASS va SCSS Sehrgarligi! 🪄

Veb-saytlarni bezashning eng qiziqarli usulini o'rganamiz

😫 1. Nega faqat CSS ning o'zi zerikarli?

Tasavvur qiling, sizda katta sayt bor. Unda 50 ta joyda "ko'k" rang ishlatilgan. Agar mijoz "Buni qizil qilaylik" desa, nima bo'ladi? Siz 50 marta kodni qidirib, o'zgartirib chiqishingiz kerak! Bu juda ko'p vaqt oladi. Shuning uchun bizga SASS kerak!

🎁 2. SASS "Variables" (O'zgaruvchilar)

Variables — bu sehrli quticha. Siz bitta qutiga rangni solib qo'yasiz va qutining nomini chaqirasiz. Qutidagi rangni bitta joydan o'zgartirsangiz, butun saytdagi ranglar avtomat o'zgaradi!

O'zingiz sinab ko'ring! 👇

Tugmalarni bosing va $primary-color o'zgarganda nimalar ro'y berishini ko'ring:

Men SASS orqali bo'yalgan qutiman!

Mening rangim ham avtomat o'zgaradi!

/* SCSS kodi */ $primary-color: #e84393; .quti { background-color: $primary-color; } .matn { color: $primary-color; }

🪆 3. SASS "Nesting" (Ichma-ich yozish)

HTML da kodlarni ichma-ich (Matryoshka o'yinchog'i kabi) yozamiz. Odatda CSS bunga ruxsat bermaydi. Lekin SASS Nesting yordamida kodni aynan HTML kabi tushunarli qilib yozish mumkin!

/* 😴 Oddiy va Uzun CSS */ nav { background: black; } nav ul { margin: 0; } nav ul li { color: white; }
/* 🤩 SASS Nesting (Juda qisqa!) */ nav { background: black; ul { margin: 0; li { color: white; } } }

🍳 4. SASS "Mixins" (Sehrli retseptlar)

Mixins — bu takrorlanadigan kodlarni bir marta yozib, xohlagan joyda chaqirib ishlatish usuli. Xuddi onangizning shirin tort retseptini daftarga yozib qo'yib, har safar kitob qidirmasdan pishirganiga o'xshaydi!

/* Retseptni yaratamiz (@mixin) */ @mixin markazga-qoyish { display: flex; justify-content: center; align-items: center; } /* Retseptni ishlatamiz (@include) */ .mening-qutim { @include markazga-qoyish; }

⚙️ 5. `.scss` fayl va Compiler (Tarjimon) nima?

Biz yozgan style.scss faylini brauzerlar (Chrome, Safari) tushunmaydi! Ular faqat oddiy CSS ni biladi. Shuning uchun bizga Compiler (tarjimon) kerak. U SASS tilidan CSS tiliga tezkor tarjima qilib beradi.
